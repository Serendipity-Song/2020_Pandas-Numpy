# 2020_Pandas-Numpy

<br>
이 리포지토리에는 인구 통계, 날씨 및 날짜와 관련된 데이터를 분석하고 시각화하기 위한 Python 코드 및 데이터 세트가 포함되어 있습니다. 
코드는 각 데이터 분석 프로젝트에 대해 별도의 파일로 구성되며 각 파일에는 코드, 문서 및 관련 데이터 세트가 있습니다.
또한 저장소에는 Matplotlib 라이브러리를 사용하여 생성된 시각화 출력이 포함되어 있습니다.
